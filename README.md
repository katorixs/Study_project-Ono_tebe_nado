# Оно тебе надо — аукцион вещей, в которые никто не верил

**Стек:** HTML, CSS (Flexbox, Grid), Figma

**О проекте:**  
Одностраничный лендинг для концептуального аукциона нестандартных лотов. В проекте сделан акцент на визуальной иерархии, типографике и точном соответствии макету.

**Особенности верстки:**
- Сетка построена на комбинации **Flexbox и Grid**
- Стилизация состояний ссылок и кнопок (:hover, :focus, :active)
- Затемнение фона через псевдоэлемент (overlay) для читаемости текста
- Pixel Perfect — точное соответствие макету из Figma
